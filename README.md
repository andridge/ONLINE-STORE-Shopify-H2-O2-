## What's included

- Remix
- Hydrogen
- Oxygen
- Shopify CLI
- React js
- Mpesa API
- ESLint
- Prettier
- GraphQL generator
- TypeScript and JavaScript flavors
- Minimal setup of components and routes

## Getting started

**Requirements:**

- Node.js version 16.14.0 or higher

Remember to update `.env` with your SHOPS DOMAIN and STOREFRONT API TOKEN!

## Building for production

```bash
npm run build
```

## Local development

```bash
npm run dev
```
